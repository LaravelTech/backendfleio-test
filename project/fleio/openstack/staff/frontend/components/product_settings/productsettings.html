Run get me a network on auto setup
This will use the OpenStack network auto allocated topology feature. You can find more information here: http://docs.openstack.org/newton/networking-guide/config-auto-allocation.html. You can also setup resources needed for the feature for each region on the configure get me a network page.

Run in regions
Select the regions for network auto allocated topology feature. In case you don't find the region you want here, make sure you run sync. Also make sure you set up auto allocated topology feature for each region as stated above.
