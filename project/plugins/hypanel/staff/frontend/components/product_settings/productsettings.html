Product configuration

Hypanel server: the server where Hypanel api is located. You can add the server here

IP count: set a default value here. It is possible to add a configurable option so users can change this value, make sure the configurable option internal name is 'hypanel_ip_count'.

OS: you need to add a configurable option with the internal name of 'hypanel_os'. Use the 'name' field from the OpenVZ Templates as input options.

Control panel: you need to add a configurable option with the internal name of 'hypanel_control_panel'. Use a dropdown with 2 option values: either 'cpanel' or 'directadmin'.

Disk size: value is expressed in GB.

Traffic: value is expressed in GB.

Memory: value is expressed in MB.

Machine type: defaults to 'openvz'. You should leave the default value.
